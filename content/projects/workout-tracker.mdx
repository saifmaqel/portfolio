### Overview

**Workout Tracker MERN App** is a full-stack fitness tracking application built using the **MERN stack**.  
It provides a secure and efficient way for users to log workouts, track progress, and manage their fitness data.

- **Authentication & Security:** Implemented **JWT** authentication with secure CRUD operations.
- **User-Specific Data:** Each user has their own workouts assigned, ensuring data separation and privacy.
- **Performance:** Integrated **serverless rate limiting** to prevent API abuse and ensure reliability.
- **Tech Stack:** **MongoDB**, **Express.js**, **React**, and **Node.js**.
- **Features:** Create, update, and delete workout logs; track personal metrics over time.
