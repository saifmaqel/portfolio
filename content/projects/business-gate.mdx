### Overview

**Business Gate** is a mobile application that allows users to create and share personalized website profiles via **QR codes**.  
The app was designed for simplicity and efficiency, enabling users to quickly generate shareable profiles while maintaining a clean and intuitive interface.

- **Frontend:** Mobile-first design using **Flutter** for cross-platform compatibility.
- **Backend:** Integrated with **PHP APIs** and **MySQL** for data storage and management.
- **Design:** UI/UX designed with **Figma**, enhanced with **Illustrator** and **Photoshop**.
- **User Experience:** Quick profile generation and QR code sharing with smooth navigation and responsive layouts.

### Achievements / Highlights

- Developed a **fully functional mobile app** capable of generating and sharing QR-based profiles.
- Implemented **secure and scalable backend APIs** with PHP and MySQL.
- Designed an **intuitive UI/UX** optimized for mobile screens.
- Ensured **cross-platform compatibility** using Flutter for iOS and Android.
- Produced **high-quality graphics and icons** using Figma, Illustrator, and Photoshop.
- Improved user engagement with **fast and reliable QR code generation**.
