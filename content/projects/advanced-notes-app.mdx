### Overview

A secure **Advanced MERN Notes App** built with **React**, **Node.js/Express**, and **MongoDB**.  
The app implements **JWT authentication**, protected routes, and middleware for API security.  
It provides users with a robust system to create, read, update, and delete notes while maintaining secure session management.

### Achievements / Highlights

- Built a **full-stack MERN application** with secure authentication.
- Implemented **JWT login/logout and token refresh** for secure session handling.
- Developed **protected API routes** using middleware to enforce access control.
- Structured the frontend with **React components** for maintainability and reusability.
- Ensured the application is **responsive** across devices.
- Followed **best practices** for REST API design, error handling, and validation.
