### Overview

**Library Management System** is a modern full-stack web application designed to manage books, users, and borrowing operations efficiently.  
It demonstrates the integration of **Next.js App Router**, **NextAuth v5**, and **Drizzle ORM** with a **PostgreSQL** backend hosted on **Neon Serverless**.

- **Fullstack:** Built with **React 19**, **Next.js 15**, **TypeScript**, and **TailwindCSS 4**.
- **Authentication:** Implemented **JWT-based login** using **NextAuth v5** and **bcryptjs**.
- **Database:** Modeled using **Drizzle ORM** with **PostgreSQL** on **Neon Serverless**.
- **Forms:** Type-safe form validation powered by **React Hook Form + Zod**.

### Achievements / Highlights

- Built a **scalable authentication system** with JWT and role-based access.
- Integrated **Drizzle ORM** for schema-safe SQL migrations and queries.
- Designed **validated forms** with clear user feedback using Zod and Sonner.
- Delivered a **clean, responsive interface** consistent with modern design standards.
